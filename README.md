# fub.*~
**Foot Switch &amp; Expression Pedal MIDI interface** has 4 of 6.35 jack.  
LEFT 2 | Foot Switch jack. (Dual Foot Switch is available.)  
RIGHT 2 | Expression Pedal jack.  

It is using ATSAND21G microcom chip as core.
